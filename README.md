## Define .env
Create your .env file in the following format:
```env
USER=
PASSWORD=
ACCOUNT=
WAREHOUSE=
DATABASE=
SCHEMA=
TARGET_TABLE=
```

## Launch build.bat
It will generate an .exe file with the program in 
> dist/main.exe
